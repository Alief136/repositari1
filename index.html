Pertemuan ke-1 ini , kita membahas tentang basic atau dasar akan git dan program-program yang bersangkutan seperti gitHub gitLab ,BitBucket , Visual Studio Code dll

Pembahasan

  • Git

    • Git adalah sistem kontrol versi terdistribusi (distributed version control) yang memungkinkan developer melacak perubahan dalam kode mereka,berkolaborasi dengan orang lain,dan mengelola berbagai versi proyek dengan berbagai fitur yang terdapat didalamnya,fitur tersebut antara lain:

    • Kontrol versi : Menyimpan riwayat perubahan , sehingga memungkinkan kita untuk kembali ke status sebelumnya jika dibutuhkan
    • Branching dan Merging : Memungkinkan kita untuk membuat cabang untuk mengembangkan fitur dan menggabungkan perubahan tersebut ke dalam codebase utama dengan merging
    • Kolaborasi : Memungkinkan kolaborasi antar developer , sehigga satu project bisa dikerjakan bersama-sama dalam waktu yang bersamaan

  • GitHub

    • GitHub merupakan suatu platform berbasis web yang digunakan untuk menyimpan dan mengelola repositori Git . Fitur utama nya antara lain :

    • Hosting repositori publik dan pribadi.
    • Alat kolaborasi seperti pull requests dan issues.
    • Integrasi dengan alat lain dan berbagai fitur manajemen proyek

  • GitLab
